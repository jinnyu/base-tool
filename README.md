# 一些基础工具类

一些常用的工具类

- 哈希接口/集合接口/字符串
- 类 (class)
- 日期
- 邮件 ( `SMTP` / `Exchange` 发送)
- http请求 (基于 `Apache HttpClient 4` 和 `OkHttp4` )
- 加解密/哈希/随机字符相关
- Id生成器
- 线程
- 条码 (条形码/二维码)
- 两步验证
